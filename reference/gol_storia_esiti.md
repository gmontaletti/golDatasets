# Storia lunga GOL: esiti 2022-2025

Serie regionale degli esiti del programma: politiche attivate (tema F),
occupazione a 60/90/180 giorni (tema H), LEP e formazione (INAPP tav
2.1), occupati totali e per percorso (INAPP tav 2.2).

## Usage

``` r
gol_storia_esiti
```

## Format

Un `data.table` long con colonne:
`data_riferimento, fonte, regione, percorso, variabile, unita, valore, confidenza, tema_storico, era, rescan_severity`.
